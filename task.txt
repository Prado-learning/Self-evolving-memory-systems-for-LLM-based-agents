我现在在研究这个方向:Self-evolving memory systems for 
LLM-based agents
📋 研究内容：探索如何让智能体在交互过程中不仅积累经验，还能自主优化其记忆系统本身，进
一步通过记忆提升智能体的能力。
🏗️  研究背景：传统的 agent 记忆系统（如简单的 RAG 或固定模板）是静态的。这带来两个问题
：一是探索效率低下，过度依赖随机尝试；二是适应性差，无法针对特定领域（如 Coding vs. 
Research）动态调整记忆权重。最新研究通过自我进化机制构建包含“自我提问（生成任务）- 
自我导航（高效探索）- 自我归因（复盘总结）”的闭环，实现 agent 在无人工标注数据情况下的
能力迭代，同时通过提炼出可复用的“工具库”与“策略集”，实现跨任务、跨模型的泛化。
🎯 研究目标：
- 调研智能体工具记忆、自我进化最新进展。
- 复现开源项目agent 在任务中自我进化的过程，提出一种针对特定任务的记忆进化策略，尝试让
 agent 自主设计出比人工预设更优的记忆检索逻辑。
- 在开源agent评估基准上验证进化带来的收益。
